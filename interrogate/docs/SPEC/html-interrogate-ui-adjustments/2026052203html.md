1. 音频采集状态优化
在音频采集小图标旁，添加采集中 / 已断开状态文字，仅显示文字，不改变原有图标样式，根据音频实时连接状态自动切换显示。
2. 视频叠加元素显隐控制
新增一个独立控制按钮，放置在开始按钮左侧；点击后一键隐藏视频上所有叠加元素（AU 强度表、右下角 4 项状态、左侧生理与姿态数据），再次点击恢复显示，不影响视频播放本身。
3. 右侧历史记录模块改造
新增折叠 / 展开切换按钮，点击可向上收起 / 展开历史记录模块，保持内部样式不变；
修改历史记录展示格式，仅保留两条最新审讯记录，每条记录只显示：审讯编号 + 简短案情摘要；
每条记录左侧添加小型头像位，当音频 / 视频开始采集时，自动截取实时人脸画面，并展示在对应审讯记录的头像位置中。
