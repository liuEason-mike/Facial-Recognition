1. 状态监测模块改造
标题修改：将原「异常波形」标题改为 **「状态监测」**，保持模块其他文字、样式不变。
布局固定：模块整体高度固定，不随内容变化而改变尺寸，避免因内容消失导致页面布局跳动。
告警状态行固定：模块内「告警状态」行位置保持固定，不受波形内容变化影响。
2. 右侧历史记录模块改造
尺寸约束：模块宽度保持不变，高度固定为仅占3 排头像高度，不额外占用空间。
折叠 / 展开功能：模块右上角添加收起 / 展开按钮，点击可将模块整体向上收起隐藏，再次点击恢复显示；折叠 / 展开过程中，头像的尺寸、排列位置不受任何影响。
头像布局：模块内仅保留 3 排紧凑排列的人脸头像，适配模块宽度，不添加其他额外内容。
