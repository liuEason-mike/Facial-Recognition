
一、未匹配到转写文本的行不展示

当前说话人分离结果中，可能存在只有时间段、只有 speaker，但没有匹配到 ASR 转写文本的记录。

前端展示时需要过滤这类记录：

1. 如果某一条结果的转写文本为空、null、undefined、空字符串，或仅包含空格，则不要渲染到页面。
2. 不展示类似以下无效内容：
   - 未匹配到转写文本
   - 暂无文本
   - 空白说话片段
3. 页面只展示“已经匹配到有效转写文本”的说话片段。
二、时间不展示到页面

当前页面中不要展示说话片段的起止时间。

三，展示格式要求

每条有效转写记录展示格式统一为：

说话人A：转写文本内容
说话人B：转写文本内容
说话人C：转写文本内容
