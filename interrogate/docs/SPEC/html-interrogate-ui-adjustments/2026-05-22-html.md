1. 左上角案情摘要模块改造
为左上角的案情摘要模块新增可折叠交互功能：添加一个折叠 / 展开切换按钮，点击可将模块信息向上收起隐藏，再次点击恢复显示，保持模块内数据与样式不变。
2. 操作按钮位置迁移
将「开始 / 继续 / 结束 / 基准检测 / 重置」按钮组，从当前视频内区域，迁移至演示视频区域的顶部水平栏（即 “仓储合同诈骗案” 的右侧），按钮的颜色、样式、尺寸与功能逻辑完全保持不变。
3. 生理与姿态数据布局调整
将左侧「生理与姿态」模块的 6 项核心数据（心率、注意力、俯仰角、偏航角、注意力），从当前左侧独立模块，迁移至检测视频画面的左侧区域展示，保持数据内容不变。
4. AU52 强度表区域改造
移除 “AU52 强度表” 文字标题；
将「设备状态」的 4 项数据（视频链路、音频链路、视频帧、音频包），调整至 AU52 数据表格的底部右侧区域展示，不破坏表格原有布局。
5. 新增麦克风标注标识
在视频画面的右侧上角区域，添加一个清晰的麦克风状态标注 / 图标，用于标识音频采集状态。
