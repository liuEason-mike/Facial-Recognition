# 当前技术债与整改要求

| 问题 | 影响 | 整改要求 |
| --- | --- | --- |
| MySQL 专用注释与 SQLite fallback 不一致 | 生产可能误用 SQLite | 生产环境缺少 MySQL 配置时必须启动失败 |
| `.env_pro` 存在敏感配置风险 | 密钥泄露风险 | 改为 `.env.example`，真实配置由部署注入 |
| `Dockerfile_backend` 存在污染字符串 | 镜像不可用 | 修复或删除该文件 |
| ASR 与 Neo4j 地址硬编码 | 环境不可迁移 | 改为环境变量 |
| `/ws/face` 成功和错误响应结构不一致 | 前端类型复杂 | 后续评估统一响应结构 |
| 依赖 `db.create_all()` | 生产迁移不可控 | 引入数据库迁移工具 |
