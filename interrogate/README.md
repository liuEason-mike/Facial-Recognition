本项目为多模态人机感知融合分析系统，集成计算机视觉、语音 ASR、自然语言处理、时序异常检测多类技术，面向实时画面 + 实时语音双数据源做全链路结构化解析，支持不定时长不间断采样分析。 核心功能模块 人脸视觉解析模块 对画面内人脸做目标框定位，精细化标注眼部、瞳孔、鼻部、嘴部人脸关键点包围框；实时采集瞳孔偏转角度、AU52 面部动作单元数据，结合面部特征综合输出实时情绪分类结果。 实时语音处理模块 接入流式音频数据流，完成实时语音转文本（ASR）；基于 NLP 算法从全量对话文本中自动提炼核心关键词、实体短语。 对话知识图谱构建模块 依托对话语义逻辑、实体关联关系，自动解析对话主谓宾逻辑、实体交互关系，结构化生成可视化对话知识图谱。 生理 \& 情绪时序异常检测模块 同步采集实时心率、情绪分值、瞳孔参数、AU 特征等多维度时序指标；不依赖固定阈值与固定检测时长，采用孤立森林 (Isolation Forest) 无监督异常检测算法，对比基线基准数据，自动判别生理指标、情绪状态的异常突变并标记告警。 应用方向 人机交互测评、心理生理监测、实时对话语义分析、多模态行为异常筛查等场景。
